# 🎬 방송용 자막 배너 생성기 Pro (Broadcast Lower-Third Generator Pro) v3.4

> 브라우저 기반으로 동작하는 고해상도 투명 방송 자막(Lower-Third) 에디터 및 실시간 투명 WebM 비디오 내보내기 도구입니다. 
> 트위치, 유튜브 스트리머, OBS 방송 송출, 영상 편집용 자막 소스 제작에 최적화되어 있습니다.

---

## 🌟 주요 기능 (Key Features)

1. **현업 방송사 수준의 15종 프리미엄 디자인 프리셋**
   * 넷플릭스 오리지널, 오스카 영화제, 다큐 인터뷰, 테크 홀로그램( scanline & 코너 브래킷), 럭셔리 에디터(Vogue), 신스웨이브, 애플 스타일 미니멀(Frosted Glass), 인스타그램 무빙 그라데이션, 긴급 속보 뉴스 등 탑재.
   * 즐겨찾기(별표) 필터링 제공.
2. **드래그 앤 드롭 & 스마트 가이드라인 자석 정렬**
   * 자막 배경 카드 및 개별 텍스트 요소들을 마우스 드래그로 자유롭게 이동.
   * 중앙/좌우 경계선에 다가갈 때 자동으로 가이드라인이 표시되며 자석처럼 달라붙는 Snap 기능 탑재.
3. **배경 카드 가로 폭(Width) 실시간 조절**
   * 전체 카드를 포커스하여 가로 너비를 20%~100% 범위로 자유롭게 늘리거나 줄일 수 있습니다.
4. **실행 취소(Undo) / 다시 실행(Redo) 지원**
   * 에디팅 중 실수했을 때 `Ctrl + Z` 단축키로 안전하게 상태 복구 가능.
5. **글리터 애니메이션(Shimmer Sweep)**
   * 자막 카드 표면을 주기적으로 훑고 지나가는 메탈릭 반사광 속도 및 폭 정밀 제어.
6. **가상 모니터 배경화면 설정**
   * 시네마틱 백드롭, 크로마키용 그린 스크린, 포토샵 레이아웃 확인용 투명 바둑판 배경 실시간 제공.
7. **투명 알파 채널 포함 WebM 동영상 다운로드**
   * 정적 `PNG` 이미지 외에, **글리터 반사광과 등장 모션이 그대로 녹화된 투명 동영상(`video/webm;codecs=vp9`)** 다운로드 지원.
   * 크로마키 작업 없이 OBS 및 프리미어 프로 등에 투명 오버레이로 즉시 사용 가능.

## 🚀 기동 방법 (How to Run)

본 프로젝트는 정적 파일(HTML/CSS/JS)로 구성되어 있어, 웹 서버(Web Server)를 통해 간편하게 로컬 호스팅하여 실행할 수 있습니다.

### 방법 1. 파이썬(Python) 내장 서버 이용 (권장)
터미널 또는 명령 프롬프트(CMD)를 열고 프로젝트 폴더로 이동한 뒤 아래 명령어를 입력합니다.

```bash
python -m http.server 9100
```
그 후 브라우저를 열고 다음 주소로 접속합니다.
👉 **[http://localhost:9100](http://localhost:9100)**

### 방법 2. Node.js `http-server` 이용
Node.js 패키지를 사용해 서버를 켭니다.

```bash
npx http-server -p 9100
```
👉 **[http://localhost:9100](http://localhost:9100)**

## 🎥 OBS Studio 연동 및 사용 팁

1. **정적 이미지 자막 사용 시**:
   * 다운로드 유형을 `투명 PNG 이미지 (정적)`로 선택 후 다운로드한 뒤, OBS의 `소스 목록` -> `+` -> `이미지`로 추가합니다.
2. **애니메이션 포함 동영상 자막 사용 시 (추천)**:
   * 다운로드 유형을 `투명 WebM 동영상 (애니메이션)`으로 다운로드합니다.
   * OBS의 `소스 목록` -> `+` -> `미디어 소스`를 생성합니다.
   * 로컬 파일을 선택하고 **[루프(반복 재생)]** 옵션을 체크하면 자막의 글리터 애니메이션이 영구적으로 반복 작동합니다.
